# -Hybrid-Transformer-Based-Sentiment-Analysis-with-Emoji-Slang-Normalization-and-Sarcasm-Detection
A hybrid sentiment analysis system for social media text that combines DistilBERT, BiLSTM, TF-IDF Logistic Regression, and RoBERTa-based sarcasm detection. The model handles emojis, slang, and contextual sentiment, achieving 86.58% accuracy on the Sentiment140 dataset.
